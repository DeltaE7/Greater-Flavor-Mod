# Greater Flavor Mod (GFM)

Este mod es HFM, pero mucho más expandido, agregando mucho contenido (provincias, cambios para mayor precisión histórica, etc).

Si eres un fan del anime y quieres gozarlo en GFM, échale un vistazo al submod de abajo!

## Cómo cambiar el idioma del juego
Para cambiar el idioma de Victoria 2 al español y jugar con la traducción, debes seguir los siguientes pasos:
1. Ve a la carpeta de datos de Victoria 2 (debería estar en Documentos/Paradox Interactive)
2. Busca la carpeta correspondiente al GFM (debes abrir el juego con el mod primero)
3. Busca el archivo [settings.txt] y ábrelo
4. En donde dice language, pon l_spanish
5. Guarda la configuración.
6. Abre el juego de nuevo; si lo hiciste bien, debería aparecer la traducción en español!

## Instrucciones para la instalación

1. Para descargar la última versión, clickea el botón verde que dice "Code" y luego "Download ZIP", o clickea [aqui](https://github.com/JmanThunder/HFM-Expanded/archive/refs/heads/master.zip)
2. Ve a la carpeta de mods de tu Victoria 2. 
  * Si tienes la versión de Steam en Windows debería estar en C://Archivos de programa/Steam/steamapps/common/Victoria2/mod
  * Si tienes una versión que no es de steam, debería estar en C://Archivos de programa/Paradox Interactive/Victoria 2/mod
3. Elimina las versiones antiguas del mod, si tienes alguna.
4. Mueve todo el contenido del ZIP dentro de la carpeta de mods.
5. Inicia el juego desde el launcher (victoria2.exe) y elige allí los mods que desees:
Debajo tienes la lista de submods que contiene el GFM y una descripción de cada uno.
Recuerda que para usar cualquiera de estos deberás abrir el juego con el GFM + el submod escogido.
  * GFM es independiente - no precisas de HFM para jugarlo.
  * GFM SW es el submod de Shattered World (mundo fragmentado) el cual divide el mapa en muchos estados pequeños que generan una experiencia random, caótica y hasta divertida!
  * GFM Low Resolution es un submod que reemplaza las imágenes del mod original con otras de menor resolución (ideal para computadores lentos)
  * GFM Dynamic Nations añade más opciones de banderas/nombres/colores para cada nación y también añade mecánicas para esto. Recomendamos no utilizar este submod si tu PC no tiene un procesador moderno y más de 8 GB de RAM, ya que es inestable.
       Nota: Es MUY recomendado instalar el Large Address Aware patch, que elimina el límite de RAM del juego por defecto.
  * GFM RGO agrega nuevos RGOs experimentales.
  * GFM Music agrega más música al juego.
7. Disfruta!

### Rendimiento mejorado (64 bits)

Para un rendimiento mejorado (PCs de 64-bits):
1. Descarga laa_2_0_4.zip de este sitio: https://www.techpowerup.com/forums/threads/large-address-aware.112556/
2. Abre el .exe
3. Selecciona V2game.exe 
4. Activa la opción "Large Address Aware Flag"
5. Guarda
6. Disfruta! El juego ahora debería irte el doble de rápido para cualquier mod.
